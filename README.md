# Nome do Projeto: IFSTARS:  Sistema de Feedbacks de alunos para professores do IFPR - Campus Foz do Iguaçu.


## 1) Identificação do projeto
- Instituição: Instituto Federal do Parána Campus Foz do Iguaçu.

- Orientadores: Ana Carla Arruda de Holanda, Daniel Di Domenico e Jefferson Chaves.

- Integrantes: Arian Moura Queiroz, Erick Jhorge Da Silva Siqueira, Heloise Ramos Figueiredo e Tailane Consorte.

- Resumo: O feedback no ambiente educacional é uma ferramenta essencial para promover melhorias no processo de ensino-aprendizagem, permitindo que os professores compreendam quais estratégias estão sendo eficazes e quais pontos podem ser aprimorados. No entanto, atualmente, no campus IFPR Foz do Iguaçu, não é utilizado um método automatizado para a realização de feedbacks dos alunos sobre as aulas dos professores. Esse processo, essencial para o aprimoramento das práticas pedagógicas, geralmente é feito manualmente ou por meio de formulários digitais, como o Google Forms, o que pode gerar dificuldades, especialmente quando os docentes não estão habituados a esse tipo de ferramenta. Visando essa problemática, o projeto desenvolvido permite oferecer um sistema em que a avaliação fosse feita digitalmente com tópicos sobre a aula de determinado professor, como metodologia e ferramentas utilizadas. Com isso os docentes teriam o feedback mais acessível e direto para compreender melhor o impacto de suas metodologias de ensino, identificando pontos de melhoria e adaptando conforme as necessidades dos alunos.

Palavras-chave: Feedback. IFPR. Professores. Alunos.
